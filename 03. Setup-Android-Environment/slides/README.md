# Setup Android
- Download SDK from [http://developer.android.com/](http://developer.android.com/sdk/index.html)
  - You need to have JDK 1.7 or higher
  - Need to have System Path `JAVA_HOME` set to `C:\Program Files\Java\jdk[version]`
